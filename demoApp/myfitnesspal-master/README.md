- Node Js
- Express
- MongoDB

## Frontend
- React
- CSS
- Styled- components

## Run Locally

Clone the project

```bash
  git clone https://github.com/sachanarpit/myfitnesspal.git
```

Go to the project directory

```bash
  cd myfitnesspal
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
